# Fanissima

The exclusive global home of women's sport.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS v4
- motion/react (spring-physics animations)
- Lucide React icons
- Radix UI primitives

## Design Language

- **Bodoni Moda** — Didone display serif for hero text & scores
- **Figtree** — warm humanist sans for UI
- **DM Mono** — numeric data labels
- Hot pink `#FF0080` + gold `#C4A35A` brand palette
- Warm purple-black `#09090F` backgrounds
- Apple Health/Fitness activity rings
- Liquid glass navigation
- iOS 27 design language

## Getting Started

```bash
npm install
npm run dev
```

## Screens

- **Home** — Live scores, breaking news, trending athletes, AI orb, community
- **Explore** — Search players, teams, competitions + follow system
- **Live** — Live matches grouped by competition
- **Me** — Profile, followed players/teams with activity rings
